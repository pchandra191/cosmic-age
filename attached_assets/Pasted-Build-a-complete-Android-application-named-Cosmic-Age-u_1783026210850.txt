Build a complete Android application named **Cosmic Age** using **Kotlin + Jetpack Compose** with a modern, premium UI and smooth animations.

## App Concept

Cosmic Age helps users understand their life in the context of the universe. The user enters their date of birth (and optionally birth time), and the app calculates detailed statistics about their age across Earth, the Solar System, and the Universe.

The design should feel like a mix of NASA, Apple, and a premium astronomy application.

---

# Technical Requirements

* Native Android App
* Kotlin
* Jetpack Compose
* Material 3
* MVVM Architecture
* Navigation Component
* DataStore for saving user profile
* Offline-first calculations
* Dark Theme by default
* Responsive UI
* Smooth animations
* Beautiful charts and visualizations
* Share statistics as image cards
* Accurate astronomical calculations using scientific constants
